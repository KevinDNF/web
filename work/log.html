Virtual Machines

In out first practical we where introduced to our Virtual Machines which we will be using for our coursework website and database design. The virtual machine was running a version of linux and we had to access it via SSH using a shell chrome extension. This was my first interaction with a linux shell I experimented with a couple of commands.

help "Displays the possible commands"

man program "Display a program's manual"

cd path "Changes directory to a path. If no path is set it changes back to the home folder."

mkdir path "Makes a directory."

ls "Lists the files and directories in the current directory. Has lots of parameters like -a which shows invisible files/directories."

One of my goals for this first year was to learn to use and how a linux os operates.

HTML 5

This week we where taught that the current html standard is HTML 5 which has been around for years. In HTML 5 every html document needs to have at the top:

<!DOCTYPE>

This ensures web browsers detect that HTML 5 is being used.

Head

Every documents needs a head section, in which important information is necessary like title, meta tags and links to stylesheets/scripts.

This informatin is not showned in the page but it is used for other necessary actions the webpage will perform.

<meta charset="utf-8">

<meta name="author" content="UP849868">

<meta name="viewport" content="width=device-width, initial-scale=1.0">

<link rel="shortcut icon" type="image/png" href="../images/logoInverted.png">

<link rel="stylesheet" href="../style.css">

The meta tag are used for metadata.

The link tag are used to link other files like stylesheets or javascript.
